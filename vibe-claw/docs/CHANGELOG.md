# Changelog

## 0.4.0

- Added stable error response fields: `code`, `message`, `details`, `requestId`.
- Added API token lifecycle metadata: expiry, IP allowlist, last-used time and IP.
- Added persisted usage counters and `/v1/usage`.
- Added billing summary and `/v1/billing`.
- Added webhook subscriptions and subscription-based run delivery.
- Added Prometheus metrics endpoint.
- Added developer docs and minimal Node/Python SDK clients.
